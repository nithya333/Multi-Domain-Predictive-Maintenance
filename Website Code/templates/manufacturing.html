Dataset Explanation

The dataset provides operational data collected from an industrial process, capturing important parameters that describe the working conditions of machinery. This data can be utilized for predictive maintenance and failure detection.

Input Features (Independent Variables):

  • UDI (Unique Identifier) - A unique ID assigned to each record.
  • Product ID - Identifier for the product being processed.
  • Type - Categorical variable indicating the type of product (L, M, or H).
  • Air Temperature [K] - The surrounding air temperature in Kelvin.
  • Process Temperature [K] - The temperature of the process in Kelvin.
  • Rotational Speed [rpm] - The rotational speed of the machinery in revolutions per minute (rpm).
  • Torque [Nm] - The torque applied to the machine in Newton-meters (Nm).
  • Tool Wear [min] - The wear level of the tool in minutes, representing its operational lifespan.

Output Features (Dependent Variables):

  • Target - Binary variable indicating whether a failure occurred:
    • 0: No Failure
    • 1: Failure
  • Failure Type - Categorical variable indicating the type of failure:
    • "No Failure": Indicates normal operation without any failure.
    • Other potential failure types (not present in this snippet) may include power failure, tool wear failure, or mechanical failure.

Model:

XGBoost Classifier

Use Case:

This dataset can be used for predictive maintenance and failure detection. By analyzing the relationship between operational parameters and failure occurrences, machine learning models can be developed to predict potential failures and optimize maintenance schedules. This helps in reducing downtime and improving efficiency.
